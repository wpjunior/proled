SAPL install

  A instalação do SAPL é bastante simples. Se você está usando
  a ferramenta CMF QuickInstaller Tool, basta ir até o objeto
  portal_quickinstaller na raiz do seu CMF Site, selecionar o
  produto ILSAPL e clicar no botão Install.

  Senão, basta criar um External Method na raiz do seu CMF Site
  com as seguintes informações:

  id: install_sapl
  Module Name: ILSAPL.install
  Function Name: install

  E clicar na aba Test desse método.

  Antes de instalar o SAPL confira o arquivo DEPENDENCIES.txt
  para certificar-se de que todas as dependências estão instaladas
  adequadamente.
